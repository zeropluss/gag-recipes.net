## 🚀 部署选项

- **Netlify** - 简单拖拽部署

### 2. 本地开发
```bash
# 启动本地服务器
python -m http.server 8000
# 或
npx http-server . -p 8000

# 访问 http://localhost:8000
```

### 3. 使用部署脚本
```bash
# Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File deploy.sh local

# Linux/Mac
./deploy.sh local
```

## 🔧 自定义指南

### 添加新菜谱
1. 在 `assets/recipes.json` 添加菜谱数据
2. 添加对应的图片到 `images/` 目录
3. 系统会自动渲染新菜谱

### 修改样式
1. **全局样式** - 编辑 `assets/tokens.css`
2. **组件样式** - 编辑 `assets/components.css`
3. **主题色彩** - 修改 CSS 变量

### 扩展功能
- **添加收藏功能** - 在 InteractionManager 中实现
- **用户评分** - 扩展 recipes.json 数据结构
- **多语言支持** - 添加国际化模块
